# Password Generator

This project is a **Password Generator Web App** that helps users create strong and customizable passwords quickly and safely.

## Features

- Choose password length using a range slider (1â€“20).
- Include or exclude character types: uppercase letters, lowercase letters, numbers, and symbols.
